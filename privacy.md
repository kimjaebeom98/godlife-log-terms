# 갓생기록 개인정보 수집·이용 동의서

**시행일: 2026년 5월 2일**

갓생로그(이하 "회사")는 「개인정보 보호법」 제15조 및 제22조에 따라 갓생기록 서비스(이하 "서비스") 이용을 위해 회원의 개인정보를 다음과 같이 수집·이용합니다. 본 동의서를 충분히 읽으신 후 동의 여부를 결정해 주세요.

## 1. 수집하는 개인정보 항목

### 가. 토스 로그인을 통해 자동 수집되는 정보

| 항목 | 설명 |
| --- | --- |
| 토스 사용자 식별키(`userKey`) | 토스 앱이 발급하는 고유 식별 번호. 회사는 이 값으로만 회원을 식별합니다. |

> 회사는 이름·전화번호·이메일·생년월일·성별·국적·연계정보(CI) 등 추가 개인정보를 토스로부터 받지 않으며, 토스 로그인 동의 화면에서도 해당 항목은 요청하지 않습니다.

### 나. 회원이 직접 입력하는 정보

| 항목 | 설명 |
| --- | --- |
| 닉네임 | 1~20자, 다른 회원에게 노출되는 식별명 |
| 한줄목표 | 50자 이내, 마이페이지·인증 상세에 노출되는 자유 문구(선택) |
| 인증 사진 | 카메라 촬영 사진. EXIF·위치정보는 업로드 전 자동 제거됩니다. |
| 수행 시간·수행 내용·해시태그·오늘의 한마디 | 인증 작성 시 입력한 텍스트 |
| 목표 정보 | 카테고리·목표명·기한·인증 주기·기본 공개 설정 |
| 좋아요·댓글 기록 | 다른 회원의 인증에 대한 상호작용 |
| 신고·차단 기록 | 부적절 콘텐츠 신고 또는 다른 회원 차단 시 발생하는 기록 |

### 다. 서비스 이용 과정에서 자동 생성·수집되는 정보

| 항목 | 설명 |
| --- | --- |
| 서비스 이용 기록 | 인증 작성 일시, 좋아요/댓글 일시, 접속 일시 |
| 신고 누적 횟수·이용 제한 상태 | 3-strike 정책 운영을 위한 집계 |

## 2. 개인정보의 수집·이용 목적

회사는 수집한 개인정보를 다음 목적에 한해서만 이용합니다.

1. **회원 식별 및 본인 확인** — 토스 사용자 식별키를 통해 동일 회원의 작성 콘텐츠를 연결
2. **서비스 핵심 기능 제공** — 목표·인증·잔디·피드·댓글·좋아요·신고·차단 기능 운영
3. **부적절 콘텐츠 모니터링 및 회원 보호** — 신고 검토, 3-strike 정책 운영, 이용 제한
4. **고객 문의 응대** — 이메일을 통한 문의·신고 접수 및 회신
5. **서비스 운영 통계 및 품질 개선** — 익명 처리된 집계 데이터 분석

## 3. 개인정보의 보유·이용 기간

원칙적으로 회원이 회원 자격을 유지하는 동안 보유·이용하며, 회원 탈퇴 시 다음 정책에 따라 처리합니다.

| 정보 | 보유 기간 |
| --- | --- |
| 회원 정보(닉네임·한줄목표·`userKey`) | 탈퇴 즉시 삭제 |
| 인증 사진·인증 내용 | 탈퇴 즉시 영구 삭제 (Storage 객체 포함) |
| 목표·잔디 데이터 | 탈퇴 즉시 삭제 |
| 좋아요·댓글 | 탈퇴 즉시 삭제 |
| 신고·이용 제한 기록 | 탈퇴 후 1년 보관 (재가입 후 회피 행위 방지 목적) |
| 관계 법령에 따른 보관 의무 정보 | 해당 법령이 정한 기간 동안 별도 보관 후 파기 |

서비스가 영구 종료되는 경우 종료일로부터 30일 후 모든 회원 정보·콘텐츠를 영구 삭제합니다(법령상 보관 의무가 있는 정보 제외).

## 4. 개인정보의 제3자 제공

회사는 원칙적으로 회원의 개인정보를 외부에 제공하지 않습니다. 다만 다음의 경우 예외로 합니다.

- 법령에 따른 수사기관·감독기관의 적법한 요청이 있는 경우
- 회원의 사전 동의가 있는 경우

## 5. 개인정보의 처리 위탁

회사는 서비스 운영을 위해 다음과 같이 개인정보 처리를 위탁하고 있습니다.

| 수탁자 | 위탁 업무 |
| --- | --- |
| Supabase Inc. | 데이터베이스·저장소(Storage) 호스팅 (Amazon Web Services 인프라 사용) |
| (주)비바리퍼블리카(토스) | 토스 앱·앱인토스 플랫폼 운영, 토스 로그인 인증, 미니앱 호스팅 |

수탁자는 위탁 목적 외 개인정보를 이용할 수 없으며, 회사는 위탁 계약을 통해 안전하게 관리되도록 감독합니다.

## 6. 동의 거부 권리 및 거부 시 불이익

회원은 본 동의서의 수집·이용에 동의하지 않을 권리가 있습니다. 다만 본 동의서의 수집 항목은 서비스 제공에 필수적인 항목이므로 동의를 거부할 경우 갓생기록 서비스를 이용할 수 없습니다.

## 7. 개인정보의 안전성 확보 조치

회사는 개인정보 보호법 제29조에 따라 다음 조치를 시행합니다.

- 모든 개인정보의 암호화된 통신(HTTPS/TLS) 전송
- 인증 사진의 EXIF 메타데이터(GPS 좌표·기기정보) 업로드 전 자동 제거
- 서버 측 접근 권한 관리 및 데이터베이스 행 단위 보안(RLS) 적용
- 직접 데이터베이스 조회 차단 및 비즈니스 로직 단일 진입점(RPC) 강제

회원은 언제든지 다음 권리를 행사할 수 있습니다.

- **열람**: 마이페이지에서 본인 정보·인증·잔디 확인
- **정정**: 마이페이지 "프로필 편집"으로 닉네임·한줄목표 변경

- **회원 탈퇴**: 토스 앱 설정 또는 회사 이메일 요청
